This project is a closed-campus peer-to-peer marketplace designed for college students to buy, sell, and exchange academic resources. Users can list items such as books, notes, lab materials, instruments, and electronics with multiple images and detailed descriptions. The platform includes category-based browsing, search and filter options, a chat system for communication, and an exchange feature that allows students to swap items instead of only selling. The goal is to promote affordability, reuse, and easy access to study materials within the campus.# campus-Mart-2
